# Тир-лист по ссылкам на игры

Небольшой веб-инструмент для создания тир-листа игр на стриме или просто для себя.  
Работает прямо в браузере, без регистрации и без установки.

Live-Version: https://tierlist-tool.vercel.app/

---

## Что умеет

- Принимает ссылки на игры (Steam, itch.io, Epic, GOG и т.д.);
- Автоматически подгружает:
  - название игры (через OpenGraph / метаданные),
  - обложку игры;
- Поддерживает:
  - добавление одной игры по URL;
  - массовое добавление сразу нескольких ссылок (по одной в строке);
- Сортировка по принципу «выбери одну из двух»:
  - показываются две игры,
  - пользователь кликает, какая нравится больше,
  - в конце получается отсортированный список;
- Автоматически строит красивую тир-таблицу (S / A / B / C / D) с обложками игр;
- Позволяет скачать тир-таблицу как картинку (PNG) для обмена или архива;
- Сохраняет прогресс локально в браузере (localStorage);
- Позволяет:
  - удалять отдельные игры,
  - перезагружать данные игры (обновить название/обложку по URL),
  - добавлять новые игры к уже готовому тир-листу без пересортировки всех старых.

---

## Как использовать

1. Открыть страницу:  
   **https://tierlist-tool.vercel.app/**
2. Ввести URL страницы игры и нажать **«Загрузить и добавить игру»**  
   или вставить сразу несколько ссылок в поле «Несколько URL».
3. После загрузки метаданных игры появятся в списке «Собранные игры».
4. Нажать **«Начать тир-лист»**:
   - откроется полноэкранный режим,
   - нужно выбирать, какая из двух игр нравится больше.
5. После окончания:
   - справа появится отсортированный список игр,
   - ниже — тир-таблица с рангами S / A / B / C / D.
6. Кнопкой **«Скачать таблицу как картинку»** можно сохранить тир-лист в PNG.

---

## Технологии

- Чистый **HTML + CSS + JavaScript** на фронтенде;
- Деплой через **Vercel**;
- Серверная функция `/api/og`:
  - принимает `?url=...`,
  - запрашивает страницу с бэкенда,
  - вытаскивает OpenGraph-метаданные (`og:title`, `og:image` и т.п.),
  - возвращает JSON с названием и картинкой.

---

## Ограничения

- Некоторые сайты могут блокировать запросы с сервера (например, отдельные страницы Steam),
  из-за чего вместо нормального названия/обложки может вернуться «Access Denied».
- В таком случае название/картинку можно задать вручную (функционал планируется/расширяется).
